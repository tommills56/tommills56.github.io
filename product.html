EnglishMaster®

EnglishMaster is a unique language learning software provided by CBT. Its greatest advantage over other learning software on the market is its real computer based test environment, huge reliable and keeping updated educational contents, and secured e-fund transfer. The system gives users the opportunity to learn English, assess their graduate study qualification through lots of practice and simulation tests, which eventually can improve users' ability and overcome their nervousness in real tests. The multi-media functions help to ensure a better learning result and a more enjoyable experience of learning. Thus, a more effective and efficient learning result can be achieved.

EnglishMaster provides users with either FREE usage mode or CHARGE usage mode. With FREE usage mode, a user can only access a very limited test content resource with limited time and functionality. But with CHARGE usage mode, a user can fully use all functionality and access all test content resources, which are continuously updated. When users login the system, they are in the FREE mode by default. All usage time within EnglishMaster is monthly based. The usage time in CHARGE mode is called charge time and the usage time in FREE mode called free time. Different usage time is counted separately.
